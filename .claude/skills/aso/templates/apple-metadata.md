## Character Limits Reference
| Field       | Limit | Notes                          |
|-------------|-------|--------------------------------|
| Title       | 30    | Include primary keyword        |
| Subtitle    | 30    | No overlap with title          |
| Promo Text  | 170   | Editable without update        |
| Keywords    | 100   | Comma-separated, no spaces     |
| Description | 4000  | Include app name 3-5x          |
| What's New  | 4000  | User-facing changes            |

## Validation Checklist
- [ ] Title ≤ 30 characters
- [ ] Subtitle ≤ 30 characters
- [ ] Keywords ≤ 100 characters
- [ ] No duplicate words across title/subtitle/keywords
- [ ] No competitor brand names in keywords
- [ ] No spaces after commas in keywords
- [ ] App name mentioned 3-5 times in description
- [ ] Description has clear structure (hook, features, benefits, CTA)
