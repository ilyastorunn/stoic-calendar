Copy-paste ready structure for Google Play Console listings.

## Title (50 characters max)
[APP TITLE - emphasize keywords, front-load important terms]

## Short Description (80 characters)
[Appears in search results - include primary keywords]

## Full Description (4000 characters)
[HOOK - Problem + Solution]

★ KEY FEATURES
✓ [Feature 1]
✓ [Feature 2]
✓ [Feature 3]

★ UNIQUE BENEFITS
• [Benefit 1]
• [Benefit 2]

★ WHAT USERS SAY
"[Testimonial]"

[CALL TO ACTION]

## Key Strategy Notes
- Keywords extracted from title and description (no separate keyword field)
- Title has highest weight in search indexing
- Front-load important terms in opening sentences
- Use ★ for section headers, ✓ and • for lists
- Avoid keyword stuffing — maintain natural language

## Validation Checklist
- [ ] Title ≤ 50 characters
- [ ] Short description ≤ 80 characters
- [ ] Full description ≤ 4000 characters
- [ ] Keywords naturally placed in first paragraph
- [ ] Clear formatting with line breaks
- [ ] Contact information included
- [ ] No keyword stuffing
