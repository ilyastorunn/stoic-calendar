## Design Specifications

### Colors
- **Background:** `#4845FF` (solid, no gradient)
- **Text:** `#FFFFFF` (white)
- **Text Subtext:** `#FFFFFF` at 90% opacity

### Typography

**Primary Font (Sans-serif):**
- Font Family: **Inter**
- Headline: Inter ExtraBold (88px, weight 800)
- Subtext: Inter Medium (36px, weight 500)
- Bullets: Inter Regular (28px, weight 400)

**Accent Font (Serif - for philosophical content only):**
- Font Family: **Cormorant Garamond**
- Usage: Screenshot 1 headline
- Size: 52px, weight 500-600

### Layout
- Letter-spacing (headlines): -1.5px
- Letter-spacing (subtext): 0px
- Line-height (headlines): 1.1
- Line-height (subtext): 1.3
- Line-height (bullets): 1.4
- Text position: Top 20-25% of screen
- Alignment: Center

## Apple App Store Requirements

- **Dimensions:** 1320 x 2868 px (portrait) for 6.9" display
- **Format:** PNG or JPEG
- **DPI:** 72
- **Quantity:** 1-10 screenshots (we're using 5)
- **File size:** No explicit limit, but keep under 10MB each
- **Content:** Must show actual app UI (no misleading features)

---

## Design Principles (North Star Aligned)

All screenshots must adhere to Stoic Calendar's North Star principles:

1. **Quiet UI** - No visual noise, clean backgrounds
2. **Truthful Visualization** - Only show real features
3. **Non-Emotional Tone** - Calm, honest messaging
4. **Widget-First** - Emphasize widgets as the primary product
5. **No Gradients** - Solid color backgrounds only (#4845FF)
